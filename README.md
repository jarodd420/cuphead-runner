# Run & Gun

A Cuphead-style left-to-right scrolling run-and-gun game. Vintage 1930s cartoon aesthetic with run, jump, and shoot gameplay.

## Controls

- **← / →** or **A / D** - Move
- **↑** or **W** - Jump
- **Z** or **Space** - Shoot

## How to Play

- Move right through the level
- Defeat enemies by shooting them
- Don't fall off or touch enemies (you'll lose a life)
- Reach the end of the level!

## Run Locally

Open `index.html` in a browser, or:

```bash
python -m http.server 8000
# or: npx serve
```

Then visit http://localhost:8000

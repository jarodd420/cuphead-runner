# Aegis AI Security — Corporate site

Static marketing site for AI security solutions: **AI Supply Chain & Asset Visibility** and **Security Gates for LLM Apps in CI/CD**.

## Run locally

**Option 1 — npm (serve)**  
From this folder:
```bash
npx --yes serve -l 3000
```
Then open: **http://localhost:3000**

**Option 2 — Python**  
```bash
python -m http.server 3000
```
Then open: **http://localhost:3000**

**Option 3 — PowerShell script**  
```powershell
.\serve.ps1
```

## Edit before going live

- Replace `contact@aegis-ai-security.example.com` in `index.html` with your real email.
- Replace "Aegis AI Security" with your company name if you prefer.
- Add your domain and any analytics/tracking if needed.
